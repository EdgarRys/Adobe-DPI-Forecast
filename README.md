# Adobe-DPI-Forecast
"The Digital Price Index is modeled after the Consumer Price Index, published by the U.S. Bureau of Labor Statistics, and uses the Fisher Price Index to track online prices."
My goal with this personal project is to use the Adobe Digital Price Index to credibly forecast the next month (Aug 2025) Core Goods Inflation excluding new and used vehicles. Please see the Summary.pdf file to review the model I selected and to view my forecast.
